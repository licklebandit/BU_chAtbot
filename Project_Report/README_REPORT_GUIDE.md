## üìê Report Formatting Guidelines

### Standard Formatting:

**Font:**
- Body text: Times New Roman, 12pt
- Headings: Bold, larger sizes (14pt, 16pt, 18pt)
- Code: Courier New or Consolas, 10pt

**Spacing:**
- Line spacing: 1.5 or Double
- Paragraph spacing: 6pt after
- Margins: 1 inch (2.54 cm) all sides

**Page Numbers:**
- Front matter: Roman numerals (i, ii, iii...)
- Main body: Arabic numerals (1, 2, 3...)
- Start Chapter 1 on page 1

**Headings:**
- Chapter titles: UPPERCASE, bold, centered
- Section headings: Title Case, bold, left-aligned
- Sub-sections: Title Case, bold, left-aligned
